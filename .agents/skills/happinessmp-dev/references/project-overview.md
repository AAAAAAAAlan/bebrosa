# HappinessMP

### Made for Windows 11

HappinessMP is developed using modern C++ to ensure compatibility and optimal performance on the latest computer systems.

### No Downgrade

HappinessMP is built for the latest version of Grand Theft Auto IV. Therefore, no downgrade is needed compared to other modifications.

### Best Synchronization

HappinessMP uses the original multiplayer network code, which brings the best possible synchronization and also allows to keep the original game AI.
